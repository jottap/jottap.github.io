## Tech Stack

The application is built on a high-performance, modern stack:

*   **Core:** [Astro](https://astro.build) - For performance and Island Architecture.
*   **Component Library:** [React](https://react.dev) - For dynamic UI components.
*   **Styling:** [Tailwind CSS](https://tailwindcss.com) - For a custom, responsive design system.
*   **Language:** TypeScript - For type safety and robust code.
